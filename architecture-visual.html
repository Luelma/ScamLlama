External API Services
🤖
Anthropic Claude API
Text / Conversation Analysis
POST /v1/messages
  • Host: api.anthropic.com
  • Model: claude-sonnet-4-20250514
  • Auth: x-api-key header
  • Key: User-provided (BYOK)
  • Payload: System prompt + conversation + context
  • Response: JSON analysis (score, patterns, recommendation)
🛡️
Reality Defender API
Deepfake Detection — Photo, Video, Audio
POST /files/aws-presigned
PUT {presignedUrl} (S3)
GET /media/users/{requestId}
  • Host: api.prd.realitydefender.xyz
  • Auth: X-API-KEY header
  • Key: Embedded (XOR) or BYOK
  • Flow: Presign → S3 Upload → Poll results
  • Polling: 3s intervals (Photo 60s / Video 120s / Audio 90s)
  • Media: JPEG, MP4, MOV, MP3, WAV, M4A, OGG, FLAC
👁️
Scam.ai API
AI Image & Video Detection
POST /ai-image-detection/detect-file
POST /video/detection
  • Host: api.scam.ai
  • Auth: x-api-key header
  • Key: Embedded (XOR) only
  • Format: Multipart form-data
  • Response: likely_ai_generated + confidence_score (0-1)
  • Mapping: ≥0.75 = FAKE, ≥0.50 = SUSPICIOUS

Backend Infrastructure Requirements
✅
Required Infrastructure
What the app depends on to operate
  • 🍎
    Apple App Store
    Distribution via App Store (ID: 6759401280). Requires active Apple Developer Program membership ($99/yr), code signing with Team ID Q73RN5N9Y2.
  • 🌐
    Anthropic API Account
    Users bring their own API key. No server-side proxy — the app calls api.anthropic.com directly from device. Usage billed to user's Anthropic account.
  • 🛡️
    Reality Defender API Account
    Embedded API key included. App makes direct calls to api.prd.realitydefender.xyz. Files uploaded to AWS S3 via RD-provided presigned URLs. Account must remain active and funded.
  • 👁️
    Scam.ai API Account
    Embedded API key included. App makes direct calls to api.scam.ai. Account must remain active and funded.
  • 📋
    GitHub Gist (Force Update)
    version.json hosted on GitHub Gist (ID: dc598476815a301faeccde950194ece9). Checked on every app launch to enforce minimum version. Free tier, no auth required.
🚫
NOT Required
Zero custom backend infrastructure
  • 🖥️
    No custom backend server
    All API calls go directly from device to third-party services. No proxy, no middleware, no API gateway.
  • 🗄️
    No server-side database
    All user data stored locally on-device via SwiftData (SQLite) and Keychain. No cloud sync, no remote backup.
  • 👤
    No user accounts or auth
    No sign-up, no login, no user management. App works immediately after consent.
  • 📊
    No analytics or telemetry
    No tracking, no crash reporting, no usage analytics. Fully private.
  • 🔄
    No push notifications
    No APNs, no server-sent events, no background refresh.
